Key Features

  • Secure Messaging: Send text messages with SHA-256 hashing to ensure message integrity.
  • File Sharing: Share images, documents, and more with your peers securely.
  • Voice Calls: Enjoy noise-reduced audio calls using WebRTC and audio filtering.
  • Location Sharing: Share your precise location with high-accuracy geolocation.
  • Peer-to-Peer: Connect directly with others using PeerJS for decentralized communication.

Techniques Used

  • SHA-256 Hashing: Verifies the integrity of messages and files to prevent tampering.
  • PeerJS: Enables peer-to-peer connections for real-time chat and calls without a central server.
  • WebRTC: Powers secure, real-time voice communication with noise reduction filters.
  • LocalStorage: Stores your messages, username, and peer list for persistence across sessions.
  • Responsive Design: Adapts to any device with CSS clamp and media queries.
